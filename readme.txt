== Installation ==

- Upload the plugin files to the /wp-content/plugins/nhrst-smartsync-table directory, or install the plugin through the WordPress plugins screen directly.
- Activate the plugin through the 'Plugins' screen in WordPress
- Use the Tools-> NHR SmartSync Table screen to see the table using api data. 
- Also a block called NHR SmartSync Table is available to display api data as table.

You're done!
